1. 什么是核销规则管理？

1、核销规则管理是对景点内门票的验票规则信息进行设置，用来关联门票，一个门票只能关联一个验票规则。
2、在核销规则设置中，可以设置规则的名称、检票通行方式、检票控制方式、验证类型、文字提示方式、最大通行次数、语音提示方式、规则开始时间、规则结束时间等核销规则信息。

2. 怎么进行核销规则管理？

2.1 添加门票核销规则

  点击【添加】按钮，在弹出的页面里录入核销规则的基本信息，其中标注为（*）的信息，为必填项。
  信息录入完成，点击【提交】按钮，保存核销规则信息。

2.2 修改门票核销规则

  在【核销规则】列表点击一条门票信息，会自动选中该规则，然后点击列表上方【修改】按钮，在弹出的对话框里录入需要修改的信息。
  信息录入完成，点击【提交】按钮，保存修改后的核销规则信息。

2.3 删除门票核销规则

  在【核销规则】列表点击一条门票信息，会自动选中该规则，然后点击列表上方【删除】按钮。
  在弹出的对话框里点击【确定】按钮，删除规则。

3. 名词解释

3.1 检票通行方式

3.2 检票控制方式

3.3 验证类型

3.4 文字提示方式

3.5 语音提示方式

3.6 规则开始时间

3.7 规则结束时间

4. 备注

1、如果核销规则已经被某个门票使用，则该规则不能进行删除操作。
