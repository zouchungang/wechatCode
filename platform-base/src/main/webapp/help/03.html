1. 什么是通道管理？

1、通道管理是对园门内通道闸机的基本信息进行设置，一个园门内可以设置多个通道闸机。
2、在通道设置中，可以设置通道闸机的通道名称、批量添加数目、进出方向（进口/出口）、是否可用（是/否）、设备当前状态(在线/离线)、验票控制方式(直接扣次数/先放行后再扣次数)等信息。

2. 怎么进行通道管理？

2.1 添加通道

  在页面左侧【景点->园门】树形列表选中一个园门。点击【添加】按钮，在弹出的页面里录入通道的基本信息，其中标注为（*）的信息，为必填项。
  信息录入完成，点击【提交】按钮，保存通道信息。

2.2 修改通道

  在【通道】列表点击一条通道信息，会自动选中该通道，然后点击列表上方【修改】按钮，在弹出的页面里录入需要修改的信息。
  信息录入完成，点击【提交】按钮，保存修改后的通道信息。

2.3 删除通道

  在【通道】列表点击一条通道信息，会自动选中该通道，然后点击列表上方【删除】按钮。
  在弹出的对话框里点击【确定】按钮，删除通道。

3. 名词解释

3.1 批量添加数目

3.2 进出方向

3.3 是否可用

3.4 设备当前状态

3.5 验票控制方式

4. 备注

1、在进行通道管理之前，需要先对园门信息进行添加。
