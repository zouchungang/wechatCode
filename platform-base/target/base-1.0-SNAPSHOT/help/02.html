1. 什么是园门管理？

1、园门管理是对景点内园门的基本信息进行设置，一个景点里可以设置多个园门。
2、在园门设置中，可以设置园门的名称、联系方式、地址、开启状态等信息。

2. 怎么进行园门管理？

2.1 添加园门

  在页面左侧【景点名称】列表选中一个景点。点击【添加】按钮，在弹出的对页面里录入园门的基本信息，其中标注为（*）的信息，为必填项。
  信息录入完成，点击【提交】按钮，保存园门信息。

2.2 修改园门

  在【园门】列表点击一条园门信息，会自动选中该园门，然后点击列表上方【修改】按钮，在弹出的页面里录入需要修改的信息。
  信息录入完成，点击【提交】按钮，保存修改后的园门信息。

2.3 删除园门

  在【园门】列表点击一条园门信息，会自动选中该园门，然后点击列表上方【删除】按钮。
  在弹出的对话框里点击【确定】按钮，删除园门。
